Interface ChiseledBookshelf

  • All Implemented Interfaces:

public interface ChiseledBookshelf
 implements Directional

    表示书架上外部可见的哪些物品槽已被占用的接口，其标记名为“slot_0_occupied”、“slot_1_occupied”等。 一个方块可能没有任何槽被占用，也可能最多只有getMaximumOccupiedSlots - 1个槽被占用。 该接口提供了方法来获取当前已占用的槽数量和位置信息。
