    Interface Description
    AmethystCluster

    Beehive'honey_level' 值代表该蜂箱中存储蜂蜜的值.
    Bell'attachment' 值表示钟是怎么固定到其他方块上的.
    BigDripleaf'tilt' indicates how far the leaf is tilted.
    BrewingStand酿造台的 'has_bottle_0', 'has_bottle_1', 'has_bottle_2' 标识接口表示是否在外部贴图渲染出瓶子.
    BubbleColumn'drag' 值表示是否存在应用到实体上的力将其移向该方块.
    Cake'bites' 数据值表示蛋糕被吃掉了多少片.

    Campfire'signal_fire' 值表示该营火是否因为底部放了干草块而冒出更多的烟 (信号营火).
    Candle'candles' represents the number of candles which are present.
    CaveVines
    CaveVinesPlant'berries' indicates whether the block has berries.
    Chain
    Chest'type' 数据值表示这个方块是大型箱子的哪一部分, 亦或是它为一个独立的箱子.
    ChiseledBookshelf表示书架上外部可见的哪些物品槽已被占用的接口，其标记名为“slot_0_occupied”、“slot_1_occupied”等。 一个方块可能没有任何槽被占用，也可能最多只有getMaximumOccupiedSlots - 1个槽被占用。 该接口提供了方法来获取当前已占用的槽数量和位置信息。
    Cocoa
    CommandBlock'conditional' 数据值表示触发此命令方块是否需要条件, 换句话说只有在此命令方块前的命令方块执行成功时才会触发本命令方块.
    Comparator'mode' 值表示该 (红石) 比较器将会以什么模式运算.
    CoralWallFan
    DaylightDetector'inverted' 值表示阳光探测器是否被调整了反转模式, 即在光照不足时再激活.
    DecoratedPot
    Dispenser与 Powerable 相似, 'triggered' 数据值表示此发射器是否被激活.

    Dripleaf
    EnderChest
    EndPortalFrame'eye' 值表示该末地传送门上是否放置了一个末影之眼 (被激活).
    Farmland'moisture' 值的的大小表示耕地距离水源 (如果有的话) .
    Fence
    Firemd_5's mixtape.
    Furnace

    GlassPane
    GlowLichen

    PitcherCrop
    PointedDripstone'thickness' represents the dripstone thickness.
    RedstoneRail铁轨的一种, 以一种或多种方式与红石交互.

    Repeater'delay' 为中继器传递信号的延迟, 即在当前信号传递过来后, 等待多少刻再激活自身以传递信号到下一个方块.

    SeaPickle'pickles' 值表示该方块有多少个海泡菜.
    Sign
    Slab'type' 值代表台阶 (/半砖) 的状态 - 上台阶, 下台阶, 或占据了整个方块的双层台阶.

    Snow'layers' 值代表该方块堆积了多少层雪片.
    Stairs'shape' 值代表阶梯方块的材质和碰撞箱的类型.
    StructureBlock'mode' 值代表结构方块可执行的不同行为模式.
    Switch
    TechnicalPiston'type' 值代表该 (技术) 活塞方块对应的类型.

    TrapDoor
    Tripwire'disarmed' 值表示该绊线被剪刀破坏, 所以不会在破坏时触发绊线钩.
